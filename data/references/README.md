# 追加資料フォルダ

このフォルダには、インタビューログの解析に使用する追加資料を配置します。

## 利用方法

1. 法制審議会の議事録やその他の参考資料を、以下のファイル形式でこのフォルダに配置してください。
2. アプリケーション実行時に、これらの資料が自動的に読み込まれ、LLMの解析に含まれます。
3. 資料が配置されていない場合でも、アプリケーションは正常に動作します。

## サポートされるファイル形式

- `.txt` - テキストファイル
- `.md` - Markdownファイル
- `.pdf` - PDFファイル
- `.docx` - Microsoft Word文書
- `.xlsx` - Microsoft Excelスプレッドシート
- `.pptx` - Microsoft PowerPointプレゼンテーション
- `.jpg`, `.jpeg` - JPEG画像（OCR処理）
- `.png` - PNG画像（OCR処理）

## 注意事項

- テキストファイル（.txt, .md）はUTF-8エンコーディングで保存してください。
- 画像ファイルからのテキスト抽出にはOCR（光学文字認識）を使用します。画質が低い場合、正確に読み取れない可能性があります。
- ファイルサイズの制限: デフォルトでは1ファイルあたり最大30MBまで読み込めます。これを超えるファイルはスキップされます。
  - この制限は環境変数 `MAX_REFERENCE_FILE_SIZE` で変更可能です（バイト単位）。
- **トークン使用量への影響**: 追加資料は各セッション分析とクロスセッション分析で繰り返し送信されます。
  - 大量の資料を配置すると、LLMのトークン消費量が大幅に増加します。
  - 推定10,000トークンを超える場合は警告が表示されます。
  - トークン数はtiktokenライブラリを使用して正確に計算されます（利用可能な場合）。
  - tiktokenが利用できない場合は概算（約4文字 = 1トークン）で計算されます。
- 必要なライブラリがインストールされていない場合、一部のファイル形式は読み込めない場合があります。
