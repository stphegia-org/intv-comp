# 追加資料フォルダ

このフォルダには、インタビューログの解析に使用する追加資料を配置します。

## 利用方法

1. 法制審議会の議事録やその他の参考資料を、テキストファイル（.txt）またはMarkdownファイル（.md）としてこのフォルダに配置してください。
2. アプリケーション実行時に、これらの資料が自動的に読み込まれ、LLMの解析に含まれます。
3. 資料が配置されていない場合でも、アプリケーションは正常に動作します。

## サポートされるファイル形式

- `.txt` - テキストファイル
- `.md` - Markdownファイル

## 注意事項

- ファイルはUTF-8エンコーディングで保存してください。
- 大きすぎるファイルはLLMのトークン制限に影響する可能性があります。
